GalaxyTours

Meu projeto tem como base realizar uma atividade somativa de front-end aonde eu tenho que resolver e completar varios desafios que esta pedindo.

🛠️ Tecnologias Utilizadas
HTML5 (Semântica e Acessibilidade)
CSS3 (Estilização e Feedback Visual)

📁 Estrutura do Projeto
O projeto foi organizado seguindo boas práticas de mercado: / ├── index.html └── src/ ├── estilos/ (Estilos da página) ├── scripts/ (Lógica de interação) └── imagens/ (Ativos visuais)

🚀 Funcionalidades Implementadas
Interatividade Visual: Elementos que reagem ao passar do mouse (Hover e manipulação de Classes).
Saudação Dinâmica: Mensagem de boas-vindas que muda conforme o horário local do usuário.
Cálculo em Tempo Real: Formulário que processa e exibe resultados assim que o usuário digital.
Injeção de Conteúdo: Adição de novos itens na tela utilizando manipulação de DOM.
